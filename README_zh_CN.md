## doubleClickFileTree 文档树双击展开（文档树增强）插件

[English](README.md)

> 当前版本：v0.1.0 首次发布

文档树双击展开子文档；

### 快速开始

- 开启插件即可；
- 其他请浏览插件设置页面（`设置`→`集市`→`已下载`→`插件`→`双击文档树`→“设置图标”），**提示：设置页可以上下滑动哦**；
- 默认禁用文档树修改图标，可到插件设置更改；
